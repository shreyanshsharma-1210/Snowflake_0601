### **Data Flow Architecture**

```mermaid
sequenceDiagram
    participant U as User
    participant F as Frontend (React)
    participant RQ as React Query
    participant API as Express API
    participant DB as Firebase
    participant ML as ML Service
    participant AI as AI Service
    
    U->>F: Interact with UI
    F->>RQ: Trigger data fetch
    RQ->>API: HTTP Request (/api/*)
    API->>DB: Query data
    DB-->>API: Return data
    API->>ML: Process with ML model
    ML-->>API: Predictions
    API->>AI: Enhance with AI
    AI-->>API: AI response
    API-->>RQ: Typed JSON response
    RQ-->>F: Update UI state
    F-->>U: Render updated view
```
